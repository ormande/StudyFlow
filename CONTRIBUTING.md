# Guia de Contribuição

Obrigado pelo interesse no **StudyFlow**!

Este é um projeto pessoal de portfólio e, no momento, **não estou aceitando contribuições de código (Pull Requests)** de terceiros. O objetivo deste repositório é demonstrar minha evolução técnica e arquitetura de software.

### Como você pode ajudar?

Embora eu não aceite código, feedbacks são muito bem-vindos! Se você encontrar um bug ou tiver uma sugestão de melhoria:

1. Vá para a aba [Issues](https://github.com/ormande/StudyFlow/issues).
2. Verifique se o problema já não foi relatado.
3. Abra uma nova Issue descrevendo o bug ou a sugestão.

Agradeço sua compreensão e apoio!
